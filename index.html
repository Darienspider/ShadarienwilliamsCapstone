D424 Vacation Planner

By Shadarien Williams

Description

The Vacation Planner mobile application is an Android app that allows users to create, manage, and track vacations and related excursions. Users can add, update, and delete vacation records, associate multiple excursions with each vacation, set date-based alerts, search stored trips, and share trip details. The app uses a local Room database for secure data storage and provides a clean, user-friendly interface for planning and organizing travel activities.
